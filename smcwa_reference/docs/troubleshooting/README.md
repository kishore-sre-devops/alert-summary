# Troubleshooting Documentation

This folder contains troubleshooting guides and diagnostic tools.

## 📋 Files

### REMOTE_TROUBLESHOOTING_GUIDE.md
Comprehensive guide on how to troubleshoot issues remotely. Includes:
- What information to collect
- Common issues and solutions
- Step-by-step diagnostic procedures
- How to share information with support

### QUICK_TROUBLESHOOTING.md
Quick reference for common issues and emergency commands.

### collect_diagnostics.sh
Automated script to collect all diagnostic information.

## 🚀 Quick Start

### If you have an issue:

1. **Quick Check**: See `QUICK_TROUBLESHOOTING.md` for common fixes

2. **Collect Diagnostics**: Run the diagnostic script
   ```bash
   cd /path/to/SMC-LAMA
   ./docs/troubleshooting/collect_diagnostics.sh
   ```

3. **Detailed Help**: Read `REMOTE_TROUBLESHOOTING_GUIDE.md` for comprehensive troubleshooting

## 📞 Getting Help

When asking for help, share:
1. What you were trying to do
2. Error messages (from browser or logs)
3. Output from diagnostic commands
4. When the issue occurred

Use the diagnostic script to collect all necessary information automatically.
